Thiết lập SFTP để tải files lên S3

Tạo một SFTP Server với AWS Transfer Family

  1. Mở bảng điều khiển Amazon Transfer Family
  • Chọn Create server.
  1. Step 1 (Choose protocols): Có 4 tùy chọn. Trong bài lab này, chúng ta chọn SFTP rồi click Next.
  2. Step 2 (Choose an identity provider): Chọn Service managed rồi click Next.
  3. Step 3 (Choose an endpoint): Chọn Publicly accessible và click Next.
  4. Step 4 (Choose a domain): Chọn Amazon S3 và click Next.
  5. Step 5 (Configure additional details): Chọn Create a new role rồi click Next.
  6. Step 6 (Review and create): Chúng ta xem lại chi tiết của SFTP Server và nhấp vào Create server
  7. Máy chủ sẽ sẵn sàng trong vài phút nữa. Như bạn thấy trạng thái (state) đang là Starting. Nhưng bạn có thể nhấp vào SFTP Server và xem máy chủ trông như thế nào (nhớ lưu lại Endpoint URL cho các bước tiếp theo).

Đến lúc này, chúng ta đã có SFTP server khởi chạy có S3 bucket làm nguồn.

Add an SFTP user

  1. Nhấp vào Add user.

  2. Bạn nhập Username, chọn IAM role, S3 bucket của bạn, dán SSH public keys. Rồi nhấp vào Add.

    • Username: ant
    • Role: sftp-access-s3 (người dùng có thể đọc từ S3)
    • Policy: None
    • Home directory:
      • Trường 1: chọn S3 bucket của bạn
      • Trường 2: bỏ trống
    • Restricted: không chọn.
    • SSH public keys: đọc hướng dẫn bên dưới.
Tạo SSH keys
  1. Với lệnh này, chúng ta tạo một cặp khóa SSH (sftp_s3 (khóa riêng tư) and sftp_s3.pub (khóa công khai)):
ssh-keygen -m PEM -f sftp_s3

Sử dụng lệnh cat sftp_s3.pub để hiển thị nội dung của khóa sftp_s3.pub. Sao chép nó và dán vào phần SSH public key.

  1. Chúng ta đã tạo thành công SFTP server với một người dùng.

Sử dụng FileZilla với SFTP Server

  1. Mở FileZilla:

    • Từ menu trên cùng, chọn File
    • Chọn Site Manager
    • Nhấp New site
    • Nhập tên sftp-s3

  2. Chọn tab General:

    • Protocol: SFTP
    • Host: Endpoint URL
    • Logon Type: Key file
    • User: ant
    • Key file: Bạn nhấn vào Browser rồi chọn đường dẫn đến nơi lưu khóa sftp_s3 key trên máy của bạn
    • Nhấn Connect.

Với cửa sổ pop-up, chọn Save passwords, rồi click OK và Connect.

Nhấn vào OK.

  1. Chúng ta đã kết nối thành công với SFTP Server với FileZilla

  2. Tải files từ máy tính của bạn lên SFTP Server.

Bạn tìm đến thư mục chứa các files, chọn chúng, nhấp phải chuột, rồi chọn Upload. 17. Các Files đã được tải lên. 18. Mở S3 bucket, bạn sẽ thấy các files mà chúng ta vừa tải lên.

  1. Xóa một file khỏi S3 bucket.

Bạn chọn một file rồi nhấn Delete.

Nhập permanently delete rồi click Delete Object.

  1. Nhấp phải chuột ở khu vực bên phải và chọn Refresh. Chỉ còn có 02 files có thể xem được.

  2. Xóa files khỏi SFTP Server bằng FileZilla. Bạn chọn 02 files, nhấp phải, rồi chọn Delete. Click Yes.

S3 bucket của bạn sẽ trống rỗng.
